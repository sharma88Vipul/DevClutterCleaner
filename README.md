# DevClutterCleaner

DevClutterCleaner is a Windows utility for developers to detect temporary files and caches generated by development tools.

## Current MVP Status

This repository currently contains the first MVP foundation:

- .NET 10 solution structure for Domain, Application, Infrastructure, WPF UI, and tests.
- NuGet global packages cache scanner.
- Recursive directory size calculation that skips inaccessible files where possible.
- Application scan service and file size formatter.
- Unit tests for scan service behavior, directory size calculation, and missing NuGet cache folders.

This version only scans and reports size. It does not delete files.

## Solution Structure

```text
src/
  DevClutterCleaner.Domain
  DevClutterCleaner.Application
  DevClutterCleaner.Infrastructure
  DevClutterCleaner.UI
tests/
  DevClutterCleaner.Application.Tests
  DevClutterCleaner.Infrastructure.Tests
```

## Requirements

- Windows
- .NET 10 SDK
- Visual Studio with .NET desktop development workload for WPF development

## Build and Test

```powershell
dotnet restore
dotnet build
dotnet test
```

## Run the UI

```powershell
dotnet run --project src/DevClutterCleaner.UI/DevClutterCleaner.UI.csproj
```

Click `Scan NuGet Cache` to show:

```text
NuGet Cache: <size>
Total reclaimable space: <size>
```

## MVP Scanner

The NuGet scanner checks the default global packages cache:

```text
%USERPROFILE%\.nuget\packages
```

If the folder is missing, the scanner returns a safe result with `Exists = false` and `SizeInBytes = 0`.
